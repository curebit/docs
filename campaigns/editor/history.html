History¶

Whenever view or stylesheet is updated editor saves changes to History which you can access from Campaign Editor navigation bar:


  1. To preview commit changes click on its link:


  2. To see commit diff open diff icon (change log will open from the right):


  3. To rollback changes preview some particular commit → save changes. Changes will be saved as a new commit.
