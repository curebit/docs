Campaigns¶

This API allows you to access campaigns.


GET /campaigns

Returns campaigns.

Parameter Description
site_slug Your Talkable Site ID. You can get this from your Talkable dashboard after you log in and create a site.

Example¶

GET https://www.talkable.com/api/v2/campaigns?site_slug=my-store&api_key=i9uil7nQgDjucCiTJu
