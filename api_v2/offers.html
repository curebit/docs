Advocate Offers¶

This API allows you to access advocate offers.


GET /offers/<id>

Returns advocate offer.

Parameter

Description

site_slug

Your Talkable Site ID. You can get this from your Talkable dashboard after you log in and create a site.

id

Offer ID obtained with origin creation.

sharing_channels

Optional: array of social sharing channels for which will be generated sharing links.

Options: facebook, twitter, linkedin, sms, other or custom, except email, reminder, facebook_sponsored and coupon.

Example¶

GET https://www.talkable.com/api/v2/offers/89238912?site_slug=my-store

Sample response:

{
  "ok": true,
  "result": {
    "offer": {
      "id": 89238912,
      "short_url_code": "dZpBwd",
      "email": "customer@example.com",
      "show_url": "https://www.talkable.com/x/iEov9g",
      "claim_url": "https://www.talkable.com/x/5B3xO1"
    }
  }
}

With sharing links¶

GET https://www.talkable.com/api/v2/offers/89238912?site_slug=my-store&sharing_channels=facebook,twitter,custom

Sample response:

{
  "ok": true,
  "result": {
    "offer": {
      "id": 89238912,
      "short_url_code": "dZpBwd",
      "email": "customer@example.com",
      "show_url": "https://www.talkable.com/x/iEov9g",
      "claim_url": "https://www.talkable.com/x/5B3xO1"
    },
    "claim_links": {
      "facebook": "https://www.talkable.com/x/8L6xO2",
      "twitter": "https://www.talkable.com/x/KB89fO",
      "custom": "https://www.talkable.com/x/Yf794w"
    }
  }
}
