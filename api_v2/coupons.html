Coupons¶

This API allows you to access and check coupons. See examples below.


GET /coupons/:code

Returns a coupon information.

Parameter

Description

site_slug

Your Talkable Site ID. You can get this from your Talkable dashboard after you log in and create a site.

code

Coupon code

 


GET /coupons/:code/permission/:email

Checks if given coupon code belongs to given person.

Parameter

Description

site_slug

Your Talkable Site ID. You can get this from your Talkable dashboard after you log in and create a site.

code

Coupon code

email

Person’s email

Example¶
