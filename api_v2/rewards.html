Rewards¶

This API allows you to access visitor rewards.


GET /rewards

Returns rewards.

Parameter

Description

site_slug

Your Talkable Site ID. You can get this from your Talkable dashboard after you log in and create a site.

visitor_uuid

Visitor uuid

status (optional)

Filter rewards by status. By default API returns only Paid rewards.

Options: Paid, Unpaid, Voided, All

Example¶

GET https://www.talkable.com/api/v2/rewards?site_slug=my-store&visitor_uuid=8a55ef82-82c1-4596-babb-e989fd717965
