Referrals¶

This API allows you to approve/void referrals.


PUT /origins/<origin_slug>/referral

Updates referral’s status.

Note

Approved or voided referrals cannot be changed to the opposite state.

Parameter Description
site_slug Your Talkable Site ID. You can get this from your Talkable dashboard after you log in and create a site.
origin_slug

If origin is a Purchase:

order_number, e.g.: "B00K2EOONI"

If origin is an Event:

event_category:event_number, e.g.: "newsletter_subscription:42"
data

JSON object with status property.

E.g. {"status": "approved"} or {"status": "voided"}

Example¶

Approve a Referral¶

curl -H "Content-Type: application/json" \
     -X PUT \
     -d '{"api_key":"i9uil7nQgDjucCiTJu","site_slug":"my-store","data":{"status":"approved"}}' \
     https://www.talkable.com/api/v2/origins/B00K2EOONI/referral
