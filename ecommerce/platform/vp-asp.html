VP-ASP 7.0 Store IntegrationÂ¶

Curebit is easily integrated into your VP-ASP 7.0 store.

  1. Open file shopthanks.asp in your chosen text editor.

  2. Locate this code: SendSmsMerchant rsorder

  3. Directly below it, please insert this code:

    Note

    To get your personal integration code, YOUR-CUREBIT-SITE-ID needs to be replaced with your Curebit Site ID. You can get this from your Curebit dashboard after you log in and create a site.

        'Begin Curebit integration code
dim strordertime, strorderdate, xcurebiturl
strordertime = rsorder("otime")
strorderdate = formatdatetime(strordertime,vblongtime)
xcurebiturl = "https://curebit.com/public/YOUR-CUREBIT-SITE-ID/purchases/create.js"
%>
<script type="text/javascript" src="<%=xcurebiturl%>?v=0.2&purchase[order_number]=<%=ordernumber%>&purchase[subtotal]=<%=getsess("OrderTotal")%>&purchase[email]=<%=GetSess("email")%>&purchase[order_date]=<%=strorderdate%>"& >
</script>
<%
'End Curebit integration code

  4. Save this file and upload to your site.

Hint

You can find additional help on the VP-ASP page.
